# Sample Hardhat Project

This project demonstrates a basic ERC20 contract, based on HardHat project structure.

```shell
npx hardhat help
npx hardhat test
REPORT_GAS=true npx hardhat test
npx hardhat node
npx hardhat run scripts/deploy.ts
```
